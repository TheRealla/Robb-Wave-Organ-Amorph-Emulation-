# Robb-Wave-Organ-Amorph-Emulation-
Cmajor code for a virtual instrument emulating the 1927 Robb Wave Organ, using asymmetric AMORPH wavetable morphing, tube saturation, and mechanical jitter. Dedicated with respect and gratitude to Claudia Anderson and the Anderson family, inspired by the disciplined best practices and order taught by Doris McInnis and Charles McInnis.
